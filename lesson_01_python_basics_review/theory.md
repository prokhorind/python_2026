# Урок 1: Повторення основ Python

## Мета уроку
Швидко повторити основні концепції Python: змінні, типи даних, умови, цикли та функції.

## Теоретичний матеріал

### 1. Змінні та типи даних

```python
# Основні типи даних
number = 42          # int (ціле число)
price = 19.99        # float (дробове число)
name = "Олексій"     # str (рядок)
is_student = True    # bool (логічний тип)
```

### 2. Умовні конструкції

```python
age = 16

if age >= 18:
    print("Повнолітній")
elif age >= 14:
    print("Підліток")
else:
    print("Дитина")
```

### 3. Цикли

```python
# Цикл for
for i in range(5):
    print(f"Число: {i}")

# Цикл while
count = 0
while count < 3:
    print(f"Лічильник: {count}")
    count += 1
```

### 4. Функції

```python
def greet(name, age=18):
    """Функція привітання"""
    return f"Привіт, {name}! Тобі {age} років."

# Виклик функції
message = greet("Марія", 16)
print(message)
```

## Ключові поняття
- **Змінна** - контейнер для зберігання даних
- **Тип даних** - вид інформації (число, текст, логічне значення)
- **Умова** - перевірка, яка повертає True або False
- **Цикл** - повторення коду певну кількість разів
- **Функція** - блок коду, який можна викликати з різними параметрами
