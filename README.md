# Python Programming Course - Data Structures

Курс програмування Python для учнів базового рівня.

## Структура курсу

### Семестр 2: Структури даних

1. **Урок 1** - Повторення основ Python (змінні, умови, цикли, функції)
2. **Урок 2** - Розв'язання задач з циклів та функцій
3. **Урок 3** - Структури даних: вступ (списки, словники, кортежі, множини)
4. **Урок 4** - Списки: методи та операції
5. **Урок 5** - Стеки (принцип LIFO)
6. **Урок 6** - Словники: методи та операції
7. **Урок 7** - Черги (принцип FIFO)
8. **Урок 8** - Вкладені структури
9. **Урок 9** - Тематична атестація

## Структура уроків

Кожен урок містить:

- **`theory.md`** - Теоретичний матеріал
- **`examples/`** - Папка з прикладами коду 
- **`practical_task/`** - Практичні завдання:
  - `task_description.md` - Опис завдань 
  - `starter_code.py` - Стартовий код з підказками

## Для учнів

### Як працювати з матеріалами:

1. **Вивчіть теорію** - прочитайте `theory.md`
2. **Розгляньте приклади** - запустіть файли з папки `examples/`
3. **Виконайте завдання** - оберіть одне з завдань у `practical_task/task_description.md`
4. **Використовуйте стартовий код** - почніть з `starter_code.py`

### Рекомендації:

- Виконуйте завдання поступово, урок за уроком
- Експериментуйте з кодом з прикладів
- Не соромтеся запитувати вчителя при виникненні питань
- Практикуйтеся регулярно

## Для вчителів

Матеріали для вчителів (розв'язки, відповіді на тести, методичні рекомендації) зберігаються окремо і не включені в цей репозиторій.

## Технічні вимоги

- Python 3.7 або новіша версія
- Текстовий редактор або IDE (рекомендується VS Code, PyCharm)
- Базові знання роботи з командним рядком

## Встановлення та запуск

1. Клонуйте репозиторій:
```bash
git clone [URL_РЕПОЗИТОРІЮ]
cd python-data-structures-course
```

2. Переконайтеся, що Python встановлено:
```bash
python --version
```

3. Запустіть приклад:
```bash
python lesson_01_python_basics_review/examples/01_variables.py
```
