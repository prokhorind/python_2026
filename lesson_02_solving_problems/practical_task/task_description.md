## Завдання 1. Інтернет-магазин

### Умова
Є список сум покупок користувача за день.

Створи функцію `count_big_orders(orders)`, яка:
- проходиться по списку за допомогою циклу `for`
- якщо сума покупки **>= 500 грн**, вважається великою
- рахує кількість великих покупок
- повертає цю кількість

У головній частині програми:
- використай готовий список `orders`
- виклич функцію
- виведи результат

### Вхідні дані
```python
orders = [120, 560, 300, 800, 150, 620]
```

### Очікуваний результат
```
Кількість великих покупок: 3
```

## Завдання 2. Соцмережа

### Умова
Соцмережа аналізує активність користувача за тиждень.

Створи функцію `analyze_likes(likes)`, яка:
- проходиться по списку за допомогою `for`
- рахує загальну кількість лайків
- рахує кількість популярних днів (лайків **>= 100**)
- повертає два значення

У головній частині програми:
- використай готовий список `likes`
- виклич функцію
- виведи результат

### Вхідні дані
```python
likes = [45, 120, 90, 200, 30, 160, 80]
```

### Очікуваний результат
```
Всього лайків: 825
Популярних днів: 3
```
