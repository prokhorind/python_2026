# Практичне завдання: Калькулятор витрат за тиждень

## Опис завдання
Створіть програму для підрахунку витрат студента за тиждень та аналізу його фінансів.

## Що потрібно зробити:

1. **Запитати ім'я студента**
2. **Ввести витрати за кожен день тижня** (7 днів)
3. **Обчислити загальні витрати** за тиждень
4. **Знайти день з найбільшими та найменшими витратами**
5. **Обчислити середні витрати** на день
6. **Підрахувати кількість днів** з витратами більше 100 грн

## Приклад роботи:

```
Як тебе звати? Олексій

Введи свої витрати за тиждень:
Понеділок: 50
Вівторок: 120
Середа: 80
Четвер: 200
П'ятниця: 150
Субота: 300
Неділя: 75

=== АНАЛІЗ ВИТРАТ ЗА ТИЖДЕНЬ ===
Студент: Олексій
Загальні витрати: 975 грн
Середні витрати на день: 139.3 грн

Найбільші витрати: Субота (300 грн)
Найменші витрати: Понеділок (50 грн)

Днів з витратами більше 100 грн: 4
```

## Час виконання: 15-20 хвилин

## Підказки:
- Використовуй змінні для кожного дня або цикл з input()
- Для знаходження мін/макс порівнюй кожне нове значення з поточним
- Середнє = загальна сума / 7
- Використовуй лічильник для підрахунку днів з великими витратами

## Бонусне завдання (для швидких):
Додай функцію, яка визначає рівень витрат:
- До 500 грн за тиждень: "Економний"
- 500-1000 грн: "Помірний"  
- Більше 1000 грн: "Високі витрати"
