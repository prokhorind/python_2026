# Практичне завдання: Перевірка правильності дужок

## Умова

Напишіть програму, яка перевіряє,
чи правильно розставлені круглі дужки у рядку.

Приклад:

"(())()" → правильно  
"(()" → неправильно  
")(" → неправильно  

---

## Алгоритм

1. Створити стек
2. Перебрати символи рядка
3. Якщо "(" → push у стек
4. Якщо ")" → 
   - якщо стек порожній → помилка
   - інакше pop
5. В кінці:
   - якщо стек порожній → правильно
   - інакше → неправильно

---

## Обмеження

Використовувати стек.
Не використовувати підрахунок кількості "(" та ")"
