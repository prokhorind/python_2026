# 🎯 Практичне завдання: Декоратор логування

## 📌 Умова

У вас є програма, яка виконує різні дії (наприклад, обчислення).

Потрібно:
1. Створити декоратор `logger`
2. Він має:
   - виводити ім'я функції
   - виводити аргументи функції
   - виводити результат

---

## 🧪 Приклад

```python
@logger
def add(a, b):
    return a + b
