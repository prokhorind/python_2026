Створи декоратор, який показує коли функція починається та закінчується.

### Що робити:
1. **Завершити декоратор** `log_function`:
   - Заповнити пропуск: `result = # TODO: виклич func з аргументами`
   - Написати: `func(*args, **kwargs)`

2. **Застосувати декоратор** до функцій:
   - Додати `@log_function` перед `say_hello(name)`

## Завдання 2: Декоратор з підрахунком
**Час: 6-8 хвилин**

Створи декоратор, який рахує скільки разів викликалась функція.

### Що робити:
1. **Завершити декоратор** `count_calls`:
   - Заповнити пропуск: `return # TODO: виклич func з аргументами`
   - Написати: `func(*args, **kwargs)`
   - Заповнити пропуск: `wrapper.count = # TODO: встанови початкове значення 0`
   - Написати: `0`

2. **Застосувати декоратор**:
   - Додати `@count_calls` перед `greet_student(name)`

### Приклад виводу:

## Завдання 3: Декоратор для перевірки
**Час: 6-8 хвилин**

Створи декоратор, який перевіряє чи всі числа позитивні.

### Що робити:
1. **Завершити декоратор** `check_positive`:
   - Заповнити пропуск: `return # TODO: виклич func з аргументами`
   - Написати: `func(*args, **kwargs)`

2. **Застосувати декоратор**:
   - Додати `@check_positive` перед `calculate_area(width, height)`
   - Додати `@check_positive` перед `calculate_average(a, b, c)`

### Приклад виводу:
```

Результат: None
```

## Підказки:
- **Декоратор**: `def my_decorator(func):`
- **Внутрішня функція**: `def wrapper(*args, **kwargs):`
- **Виклик функції**: `result = func(*args, **kwargs)`
- **Застосування**: `@my_decorator` перед функцією
- **Підрахунок**: `wrapper.count = 0` (до return wrapper)
- **Перевірка позитивних**: `all(x > 0 for x in args if isinstance(x, (int, float)))`
