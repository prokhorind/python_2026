# 🧪 Самостійна робота: Структури даних 

## 🎯 Мета:
Навчитися обирати правильну структуру даних для задачі та працювати з нею.

---

## 📌 Умова задачі

У вас є система керування практичними роботами для учнів.

На початку вже є список практичних робіт:
- "Практична 1: Змінні"
- "Практична 2: Умови"
- "Практична 3: Цикли"

Учні виконують роботи **у порядку їх додавання**.

### 🔹 Завдання:
1. Оберіть відповідну структуру даних для зберігання практичних робіт.
2. Реалізуйте програму, яка:
   - має початкові 3 роботи
   - дозволяє додати нову роботу (метод `enqueue`)
   - дозволяє обробити всі роботи (метод `poll`)

### 📥 Ввід:
Користувач вводить нову практичну роботу.

### 📤 Вивід:
Вивести всі роботи у порядку їх виконання.

---

## Додаткове завдання 
Дозвольте користувачу додати кілька практичних (через цикл)
Додайте перевірку: якщо черга порожня — вивести повідомлення
Реалізуйте меню:
1 — додати роботу
2 — обробити роботи
3 — вийти
