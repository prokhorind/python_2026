## Завдання 1: Розклад уроків (Легке)
**Час виконання:** 15 хвилин

### Опис
Створіть систему для зберігання та відображення розкладу уроків класу на тиждень.

### Що потрібно зробити:
1. Створіть словник з розкладом на тиждень (понеділок-п'ятниця)
2. Кожен день містить список уроків
3. Додайте функцію для показу розкладу на конкретний день
4. Додайте функцію для показу всього розкладу
5. Додайте функцію для підрахунку загальної кількості уроків на тиждень

### Структура даних:
```python
schedule = {
    "понеділок": ["математика", "фізика", "українська", "англійська"],
    "вівторок": ["хімія", "біологія", "історія", "фізкультура"],
    # ... інші дні
}
```

### Приклад роботи:
```
Розклад на понеділок:
1. математика
2. фізика
3. українська
4. англійська

Загальна кількість уроків на тиждень: 20
```

### Додаткове завдання:
- Знайдіть день з найбільшою кількістю уроків

## Завдання 3: Система оцінок класу (Складне)
**Час виконання:** 20 хвилин

### Опис
Створіть повну систему обліку оцінок для класу з кількома предметами та учнями.

### Що потрібно зробити:
1. Створіть список учнів, де кожен учень - це словник
2. Кожен учень має ім'я та словник оцінок по предметах
3. Додайте функцію для обчислення середнього балу учня
4. Додайте функцію для знаходження кращого учня класу
5. Додайте функцію для обчислення середнього балу по предмету
6. Покажіть топ-3 учнів класу

### Структура даних:
```python
students = [
    {
        "name": "Анна Петренко",
        "grades": {
            "математика": [12, 11, 10, 12],
            "фізика": [11, 10, 11, 12],
            "хімія": [10, 11, 10, 11]
        }
    },
    # ... інші учні
]
```

### Приклад роботи:
```
Середній бал Анни Петренко: 11.0

Топ-3 учні класу:
1. Віра Сидоренко - 11.8
2. Анна Петренко - 11.0
3. Борис Іваненко - 10.2

Середній бал по математиці: 10.5
```

### Додаткове завдання:
- Знайдіть предмет з найвищим середнім балом у класі
- Додайте функцію для додавання нової оцінки учню

---

## Критерії оцінювання:
- **Відмінно (12 балів):** Завдання виконано повністю + додаткове завдання
- **Добре (9-11 балів):** Завдання виконано повністю
- **Задовільно (6-8 балів):** Завдання виконано частково
- **Потребує доопрацювання (менше 6 балів):** Завдання не працює або має серйозні помилки

## Підказки:
1. **Для роботи зі словниками:**
   - `dict.keys()` - отримати всі ключі
   - `dict.values()` - отримати всі значення
   - `dict.items()` - отримати пари ключ-значення

2. **Для роботи зі списками словників:**
   - Використовуйте цикли `for` для перебору
   - `len(list)` для підрахунку кількості елементів

3. **Для обчислень:**
   - `sum(numbers) / len(numbers)` для середнього значення
   - `max(numbers)` та `min(numbers)` для екстремумів

4. **Для сортування:**
   - `sorted(list, key=lambda x: x['field'])` для сортування списку словників

5. **Приклад доступу до вкладених даних:**
