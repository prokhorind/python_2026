# Lambda функції в Python

## 🔹 Що таке lambda функції?

Lambda функція — це **анонімна (безіменна) функція**, яка записується в один рядок.

Синтаксис:
lambda аргументи: вираз

Приклад:
square = lambda x: x * x
print(square(5))  # 25

---

## 🔹 Де використовуються?

Найчастіше разом з:
- map()
- filter()
- sorted()
- key у функціях

---

## 🔹 Функціональне програмування

Функціональне програмування — це підхід, де:
- функції можна передавати як аргументи
- функції повертають інші функції
- мінімізується зміна стану (immutable дані)

---

## 🔹 Основні інструменти FP у Python

- lambda
- map()
- filter()
- reduce() (з модуля functools)

Приклад:

numbers = [1, 2, 3, 4]

# звичайно
result = []
for n in numbers:
    result.append(n * 2)

# функціонально
result = list(map(lambda n: n * 2, numbers))

---

## 🔹 Порівняння підходів

### 1. Процедурний стиль

def square(x):
    return x * x

✔ Простий  
✔ Зрозумілий  
❌ Більше коду  

---

### 2. Функціональний стиль (lambda)

square = lambda x: x * x

✔ Короткий запис  
✔ Зручно для одноразових функцій  
❌ Гірша читабельність при складній логіці  
❌ Не підходить для великих функцій  

---

### 3. ООП підхід

class Math:
    def square(self, x):
        return x * x

✔ Добре для великих систем  
✔ Інкапсуляція логіки  
❌ Надлишковий для простих задач  

---

## 🔹 Переваги lambda

- Менше коду
- Зручно для швидких операцій
- Добре працює з колекціями

---

## 🔹 Недоліки lambda

- Лише один вираз
- Не можна писати складну логіку
- Важче читати для новачків

---

## 🔹 Коли використовувати?

✔ Коли функція проста  
✔ Коли використовується один раз  
✔ В map/filter/sorted  

❌ НЕ використовувати:
- для складної логіки
- для великих функцій
