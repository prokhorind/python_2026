# Deque у Python

## Що таке Deque

Deque (Double Ended Queue) — це структура даних, яка дозволяє додавати і видаляти елементи з обох сторін.

Можна:
- додавати елементи справа
- додавати елементи зліва
- видаляти елементи справа
- видаляти елементи зліва

Тобто deque — це черга, яка працює з двох кінців.

У Python deque знаходиться у модулі collections.

import collections

collections.deque

---

# Чому deque потрібен

Якщо використовувати звичайний список Python (list) для черги, виникає проблема.

Приклад:

queue = [10, 20, 30, 40]

queue.pop(0)

Після видалення першого елемента всі інші елементи зміщуються.

Було

[10, 20, 30, 40]

Стало

[20, 30, 40]

Це відбувається тому що list у Python — це dynamic array.

Видалення з початку має складність O(n).

Тому для великих черг це працює повільно.

---

# Як реалізовано deque під капотом

Deque у Python реалізований не як звичайний linked list.

Він реалізований як двозв'язаний список блоків.

Тобто структура виглядає приблизно так:

block <-> block <-> block

Кожен блок містить невеликий масив елементів.

---

# Структура одного блоку

Кожен блок містить:

- масив елементів
- посилання на попередній блок
- посилання на наступний блок

Схематично:

block

[ x x x x x x x x ]

↓

block

[ x x x x x x x x ]

↓

block

[ x x x x x x x x ]

---

# Чому використовуються блоки

Якби deque був звичайним linked list:

[10] -> [20] -> [30]

кожен елемент мав би:

- об'єкт
- два посилання

Це займає багато пам'яті.

Тому Python використовує блоки з кількома елементами.

Це дозволяє:
- економити пам'ять
- швидше працювати з кешем процесора
- швидко додавати елементи

---

# Як працює append()

Новий елемент додається у правий блок.

Наприклад:

[10 20 30 _]

append(40)

[10 20 30 40]

Якщо блок заповнений, створюється новий блок справа.

---

# Як працює appendleft()

Елемент додається у лівий блок.

Якщо місця немає, створюється новий блок зліва.

---

# Як працює popleft()

Елемент просто забирається з лівого блоку.

Було

[10 20 30 40]

popleft()

Стало

[20 30 40]

Жодного зсуву елементів не відбувається.

---

# Складність операцій

append() працює за O(1)

appendleft() працює за O(1)

pop() працює за O(1)

popleft() працює за O(1)

Тому deque набагато ефективніший для черг ніж list.

---

# Приклад використання

from collections import deque

queue = deque()

queue.append("Alice")
queue.append("Bob")
queue.append("Charlie")

print(queue)

first = queue.popleft()

print(first)

print(queue)

---

# Де використовується deque

Deque часто використовують у:

- алгоритмах BFS
- чергах задач
- обробці запитів сервера
- потоковій обробці даних
- буферах

---

# Висновок

Deque — це спеціальна структура даних у Python для ефективної роботи з чергами.

Вона реалізована як двозв'язаний список блоків.

Завдяки цьому додавання і видалення елементів з обох сторін працює за O(1).
