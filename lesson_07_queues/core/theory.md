# Queue (Черга) у Python

## Що таке Queue

Queue (черга) — це структура даних, яка працює за принципом:

FIFO — First In First Out  
(перший прийшов — перший пішов)

Приклад з життя — черга в магазині:

[Анна] -> [Богдан] -> [Катя]

Анна прийшла першою → Анна обслуговується першою.

---

# Основні операції черги

Операція | Опис
--- | ---
enqueue(element) | додати елемент у кінець черги
dequeue() | забрати елемент з початку черги
peek() | подивитися перший елемент
is_empty() | перевірити чи черга порожня
size() | кількість елементів

---

# На яких структурах даних будують Queue

Чергу можна реалізувати на різних структурах даних.

---

# 1. Dynamic Array (Python list)

У Python найпростіше реалізувати чергу через list.

Приклад структури:

[10, 20, 30]

Перший елемент видаляється з початку.

Основні операції:

enqueue → append()

dequeue → pop(0)

Проблема такої реалізації:

pop(0) має складність O(n), тому що всі елементи після першого зсуваються вліво.

Приклад:

Було:

[10, 20, 30, 40]

Після pop(0):

[20, 30, 40]

Всі елементи змістилися.

---

# 2. Linked List

Чергу можна реалізувати через зв'язаний список.

Структура:

[10] -> [20] -> [30]

Кожен елемент (node) містить:
- значення
- посилання на наступний елемент

Перевага:

enqueue → O(1)  
dequeue → O(1)

Бо не потрібно зсувати всі елементи.

---

# 3. Deque (Double Ended Queue)

У Python існує спеціальна структура для черг:

collections.deque

Deque означає **double ended queue** — черга з двох сторін.

Структура виглядає так:

[10] <-> [20] <-> [30]

Це **двозв'язаний список блоків**, тому операції виконуються дуже швидко.

Основні методи:

append() — додати справа  
appendleft() — додати зліва  
pop() — забрати справа  
popleft() — забрати зліва

Всі ці операції працюють за **O(1)**.

---

# Порівняння реалізацій

Реалізація | Структура | enqueue | dequeue
--- | --- | --- | ---
list | dynamic array | O(1) | O(n)
linked list | nodes | O(1) | O(1)
deque | doubly linked blocks | O(1) | O(1)

---

# Висновок

Чергу можна реалізувати різними способами.

Для навчання зручно використовувати list.

Для реальних програм у Python найчастіше використовують **collections.deque**, тому що всі операції виконуються за O(1).
