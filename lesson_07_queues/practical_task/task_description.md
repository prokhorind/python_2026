# Практичні завдання: Черги

## Інструкції
Оберіть **одне** з трьох завдань для виконання. Кожне завдання розраховане на 15-20 хвилин роботи.

---

## Завдання 1: Черга в їдальні (Легке)
**Час виконання:** 15 хвилин

### Опис
Створіть систему черги для шкільної їдальні, де учні стають в чергу за обідом.

### Що потрібно зробити:
1. Створіть чергу для учнів
2. Додайте функцію для додавання учня до черги (з ім'ям та класом)
3. Додайте функцію для обслуговування наступного учня
4. Покажіть, скільки учнів залишилось в черзі
5. Протестуйте з 5-6 учнями

### Приклад роботи:
```
До черги приєднався: Анна (7-А клас)
До черги приєднався: Борис (8-Б клас)
Черга: ['Анна (7-А)', 'Борис (8-Б)']

Обслужили: Анна (7-А клас)
Залишилось в черзі: 1 учень
```

### Додаткове завдання:
- Додайте підрахунок загальної кількості обслужених учнів

---

## Завдання 2: Черга завдань (Середнє)
**Час виконання:** 18 хвилин

### Опис
Створіть систему черги завдань для комп'ютера, де програми додають завдання для виконання.

### Що потрібно зробити:
1. Створіть чергу завдань
2. Кожне завдання має назву та час виконання (в секундах)
3. Додайте функцію для додавання завдання
4. Додайте функцію для виконання наступного завдання
5. Покажіть загальний час всіх завдань в черзі
6. Протестуйте з різними завданнями

### Приклад роботи:
```
Додано завдання: Копіювання файлів (30 сек)
Додано завдання: Сканування вірусів (120 сек)
Загальний час черги: 150 секунд

Виконуємо: Копіювання файлів (30 сек)
Завдання виконано за 30 секунд
Залишилось завдань: 1
```

### Додаткове завдання:
- Додайте можливість скасування завдання за назвою

---

## Завдання 3: Система повідомлень (Складне)
**Час виконання:** 20 хвилин

### Опис
Створіть систему черги повідомлень для чат-додатку, де повідомлення обробляються по черзі.

### Що потрібно зробити:
1. Створіть чергу повідомлень
2. Кожне повідомлення має відправника, текст та час
3. Додайте функцію для надсилання повідомлення
4. Додайте функцію для обробки наступного повідомлення
5. Покажіть статистику: скільки повідомлень оброблено та скільки в черзі
6. Додайте можливість перегляду черги без обробки

### Приклад роботи:
```
Повідомлення від Анна: Привіт! (14:30:15)
Повідомлення від Борис: Як справи? (14:30:20)
В черзі: 2 повідомлення

Обробляємо: Анна -> "Привіт!" (14:30:15)
Повідомлення доставлено
Оброблено: 1, В черзі: 1
```

### Додаткове завдання:
- Додайте фільтрацію повідомлень за відправником
- Покажіть час очікування кожного повідомлення

---

## Критерії оцінювання:
- **Відмінно (12 балів):** Завдання виконано повністю + додаткове завдання
- **Добре (9-11 балів):** Завдання виконано повністю
- **Задовільно (6-8 балів):** Завдання виконано частково
- **Потребує доопрацювання (менше 6 балів):** Завдання не працює або має серйозні помилки

## Підказки:
1. Використовуйте `from collections import deque` для ефективної черги
2. Або створіть чергу зі звичайного списку: `queue = []`
3. Для додавання: `queue.append(item)` 
4. Для забирання: `queue.popleft()` (deque) або `queue.pop(0)` (список)
5. Перевіряйте, чи черга не порожня перед забиранням елементів
6. Використовуйте `datetime.now().strftime("%H:%M:%S")` для часу
