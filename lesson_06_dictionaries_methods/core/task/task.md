# Практичне завдання: Реєстр книг бібліотеки

## Умова

Створіть програму, яка зберігає книги бібліотеки у dictionary.

Ключ — назва книги.
Значення — кількість примірників.

Програма повинна:

1. Дозволяти додати книгу
2. Збільшувати кількість, якщо книга вже існує
3. Виводити всі книги
4. Показувати кількість конкретної книги
5. Видаляти книгу, якщо кількість стала 0
